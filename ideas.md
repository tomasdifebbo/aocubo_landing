# Ideias de Design - AoCubo Landing Page

## Abordagem Escolhida: Minimalismo Sofisticado com Elegância Imobiliária

**Design Movement:** Minimalismo Contemporâneo com influências de Luxury Branding

**Core Principles:**
1. **Espaço Generoso:** Amplo uso de whitespace para criar uma sensação de sofisticação e respiro visual
2. **Tipografia Estratégica:** Contraste entre Domine (serifada, elegante) para títulos e Inter (sans-serif, moderna) para corpo
3. **Paleta Restrita:** Uso controlado de cores - dourado/bege como accent, preto/branco como base
4. **Profundidade Sutil:** Sombras leves e transições suaves para criar dimensão sem poluição visual

**Color Philosophy:**
- **Dourado (#D2B589):** Transmite exclusividade, sofisticação e confiança no mercado imobiliário premium
- **Preto/Cinza Escuro (#09090B):** Elegância e profissionalismo
- **Branco (#FFFFFF):** Clareza, abertura e confiança
- **Bege/Creme:** Tons quentes que humanizam a interface sem perder sofisticação

**Layout Paradigm:**
- Hero section com imagem de fundo arquitetônico (prédio moderno)
- Grid assimétrico para cards de imóveis (não uniformemente distribuído)
- Seções alternadas entre fundo branco e bege claro
- Navegação minimalista no topo

**Signature Elements:**
1. **Linha Dourada Divisora:** Elemento visual que separa seções com elegância
2. **Cards com Hover Elevado:** Efeito de profundidade ao passar o mouse
3. **Tipografia em Camadas:** Títulos grandes em Domine, subtítulos em Inter com peso variado

**Interaction Philosophy:**
- Transições suaves (0.2s cubic-bezier)
- Hover effects que elevam elementos sem exagero
- Feedback visual claro mas discreto

**Animation:**
- Fade-in suave ao carregar a página
- Transições de cor em botões (dourado → dourado mais escuro)
- Elevação de cards com sombra crescente

**Typography System:**
- **Display:** Domine, 48-64px, weight 400, para títulos principais
- **Heading:** Domine, 32-36px, weight 400, para seções
- **Body:** Inter, 16px, weight 400, para parágrafos
- **Small:** Inter, 14px, weight 400, para labels e informações secundárias
